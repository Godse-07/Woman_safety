# Safe Circle - Women’s Safety App

Safe Circle is a Flutter-based women’s safety application focused on faster help, safer travel, and better family communication.

## Key Features

- **Live Location Sharing**
  - Share your live location with trusted contacts for real-time safety tracking.
- **Family Chat (Parents)**
  - Stay connected with parents using built-in chat.
- **Nearby Essential Services**
  - Quickly find police stations, hospitals, pharmacies, and bus stops.
- **Community Chat**
  - Share experiences, safety tips, and support with other users.

## Getting Started (Flutter)

### Prerequisites

- Flutter SDK installed
- Dart SDK (bundled with Flutter)
- Android Studio / VS Code with Flutter extension
- Android Emulator or physical device

### Run Locally

1. Clone the repository.
2. Open the project folder.
3. Install dependencies:

   ```bash
   flutter pub get
   ```

4. Run the app:

   ```bash
   flutter run
   ```

### Helpful Commands

```bash
flutter analyze      # Static analysis
flutter test         # Run tests
flutter clean        # Clean build artifacts
```
